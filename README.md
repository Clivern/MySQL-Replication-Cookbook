HAProxy-Cookbook
================

A Chef Cookbook To Install and Manage [HAProxy](https://www.consul.io/).

Usage
-----

1. Add the following recipes to your node run list.

```bash
{
    "run_list": [
        "recipe[haproxy-cookbook::default]"
    ]
}
```
